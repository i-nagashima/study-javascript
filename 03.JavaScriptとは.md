# JavaScript と ECMAScript

ECMAScript =  
JavaScript 言語のコアの部分を ECMAScript として仕様策定

ブラウザ環境の場合、JavaScript の中に  
　 ECMAScript  
　 Web APIs(DOM API...) // ブラウザの場合  
がある

Node.js 環境の場合、JavaScript の中に  
　 ECMAScript  
　 CommonJS  
がある
