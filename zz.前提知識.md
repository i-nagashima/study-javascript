# オブジェクト

名前(プロパティ, またはキー)と値(バリュー)をペアで管理する入れ物

```js
let obj = {
  property1: 'Hello',
  property2: function () {},
  property3: {
    d: 'Bye',
  },
};

// プロパティへのアクセス方法
// ドット記法
obj.name;
// ブラケット記法
obj['name'];
```
